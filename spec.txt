MOUSE ACTIONS
-------------

RIGHT CLICK to upvote.
Hold RIGHT CLICK, then press LEFT CLICK to downvote.
Hold RIGHT CLICK, then move mouse N/S/E/W to use specified actions for those directions.

User menu:
			TAG
REPORT				MESSAGE
			FOLLOW

Comment menu:
			AWARD
REPORT				REPLY
			SAVE

Post menu:
			AWARD
REPORT				COMMENT
			SAVE

DOUBLE LEFT CLICK to collapse comment trees.
